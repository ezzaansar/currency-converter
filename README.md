# Currency Converter

A simple web-based currency converter that fetches real-time exchange rates and displays country flags for selected currencies.

## Features

- Convert between 160+ currencies
- Real-time exchange rates from public API
- Dynamic country flag display
- Clean, responsive UI

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- [Currency API](https://github.com/fawazahmed0/currency-api) for exchange rates
- [Flags API](https://flagsapi.com/) for country flags

## Usage

1. Open `currency converter/index.html` in a browser
2. Enter the amount to convert
3. Select source and target currencies
4. Click "Get Exchange Rate"
